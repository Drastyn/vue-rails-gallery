# vue-laravel-images

A simply practice Gallery maked with VueJS and Laravel 8 (API Rest).

## Project setup
```
npm install

(
    AXIOS,
    NODE SASS,
    BULMA CSS,
    VUE-FLASH-MESSAGES,
    FONTAWESOME
)

```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
